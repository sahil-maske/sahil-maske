## 🚀 Featured Project — PeerLearn

> **Teach what you know. Learn what you don't.**

A peer-to-peer skill exchange platform for developers — built from scratch, shipped in production.

```
🔐 Firebase Auth          ✅ Complete
💬 Real-time Chat         ✅ Complete
⌨️  Typing Indicators      ✅ Complete
✅  Read Receipts          ✅ Complete

🖼️  Profile Photos         ✅ Complete (Cloudinary)
🤝 Peer Matching          ✅ Complete
```
